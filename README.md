PAINT ENGINE - Engine para jogos de pintar

*** CONSTRUCTOR

    new PaintEngine( { arg_obj } ) 

*** COLOR (SETTER)

    PaintEngine.color = {r:,g:,b:a:}

*** { arg_obj }

Requeridos:
- sketch_canvas: main canvas id
- sketch_files: array of sketchs filenames

Opcionais:
- color: color object ({r:,g:,b:,a:})
